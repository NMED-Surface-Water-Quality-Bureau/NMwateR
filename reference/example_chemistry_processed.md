# Example Chemistry Processed Table

Example dataset of processed chemistry results for water quality
analysis.

## Usage

``` r
example_chemistry_processed
```

## Format

A data frame with 5,144 rows and 21 columns:

- WATER_ID:

  Assessment Unit identifier

- WATER_NAME:

  Assessment Unit name

- PROJECT_NAME:

  Project name

- STATION:

  Monitoring station code

- STATION_NAME:

  Monitoring station name

- SAMPLING_EVENT_TYPE:

  Type of sampling event

- DATE:

  Date of sample collection

- TIME:

  Time of sample collection

- ACTIVITY_TYPE:

  Type of activity performed

- SAMPLE_TYPE:

  Type of sample collected

- FLT_10UG:

  Filter size indicator (10 Âµg)

- SAMPLE_FRACTION:

  Fraction of sample analyzed (e.g., dissolved, total)

- CHR_UID:

  Characteristic unique identifier

- CHARACTERISTIC_NAME:

  Name of measured characteristic (e.g., nitrate, pH)

- MEASUREMENT:

  Reported measurement value (as text)

- MEASUREMENT_num:

  Numeric measurement value

- UNITS:

  Units of measurement

- LESS_THAN_YN:

  Indicator if value is less than detection limit (Y/N)

- ASSESSABILITY_QUALIFIER_CODE:

  Qualifier for assessability

- ACT_START_DATE:

  Activity start date

- ACT_END_DATE:

  Activity end date
