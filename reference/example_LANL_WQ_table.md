# Example LANL WQ Table

Example dataset containing LANL collected water quality data.

## Usage

``` r
example_LANL_WQ_table
```

## Format

A data frame with 15 rows and 900 columns:

- AU ID:

  Unique identifier for the Assessment Unit

- AU / Waterbody Name:

  Name of the waterbody or AU

- Submitter / Data Source:

  Organization or individual that submitted the data

- Station ID:

  Unique identifier for the monitoring station

- Station Name:

  Descriptive name of the monitoring station

- Sample Date and Time:

  Date and time when the sample was collected

- Sample Media:

  Type of sample collected

- Sample Fraction:

  Portion of the sample analyzed

- Parameter Name:

  Name of the analyte

- Reported Value:

  Numeric result of the analysis

- Unit:

  Measurement unit for the reported value

- Sample Detection Limit (SDL):

  Minimum concentration that can be reliably detected by the analytical
  method

- SDL Units:

  Units associated with the detection limit

- Less than detection limit?:

  Indicator if the result is below the detection limit

- Analytical Method:

  Method or procedure used for analysis

- Dilution Factor:

  Factor by which the sample was diluted prior to analysis

- Lab Name:

  Name of the laboratory that performed the analysis

- Analysis Date and Time:

  Date and time when the analysis was completed

- Lab Qualifier Code:

  Laboratory-provided qualifier indicating data quality or conditions

- Submitter Qualifier Code:

  Submitter-provided qualifier for additional context or data quality
  notes

- SSWQC Copper Acute (µg/L):

  State Surface Water Quality Criterion for acute copper toxicity

- SSWQC Copper Chronic (µg/L):

  State Surface Water Quality Criterion for chronic copper toxicity
