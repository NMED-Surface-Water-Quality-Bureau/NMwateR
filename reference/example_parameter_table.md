# Example Parameter Table

Example dataset containing parameters for SQUID analysis.

## Usage

``` r
example_parameter_table
```

## Format

A data frame with 133 rows and 17 columns:

- CHR_UID:

  Characteristic unique ID

- CHR_NAME:

  Characteristic name

- WQS_NAME:

  Water quality standard name

- SortOrder:

  Sort order for display

- ShortName:

  Short name for parameter

- ShortNameOLD:

  Previous short name

- Units:

  Units of measurement

- USE:

  Designated use category

- ATTAINS_NAME:

  ATTAINS name

- ATTAINs_GROUP_ID:

  ATTAINS group identifier

- ATTAINS_GROUP_NAME:

  ATTAINS group name

- ATTAINS_ID:

  ATTAINS identifier

- ADB_NO:

  ADB number

- ADB_NAME:

  ADB name

- CAS:

  CAS number

- STORET:

  STORET code

- SHORT_CAUSE_NAME:

  Short cause name
