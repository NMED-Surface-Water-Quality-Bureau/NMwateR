# Example SQUID Lake Profile Table

Example dataset for SQUID lake profile analysis.

## Usage

``` r
example_SQUID_LakeProfile_table
```

## Format

A data frame with 644 rows and 14 columns:

- PROJECT_NAME:

  Project name

- WATER_ID:

  Waterbody identifier

- MLOC_NAME:

  Monitoring location name

- MLOC_ID:

  Monitoring location ID

- ACT_ID:

  Activity identifier

- RES_DEPTH_HEIGHT:

  Depth or height of measurement

- CHARACTERISTIC:

  Characteristic measured

- MEASUREMENT:

  Measurement value

- UNITS:

  Units of measurement

- SE_START_DATE_TIME:

  Sampling event start date/time

- SWQB_QUALIFIER_CODE:

  Qualifier code for SWQB

- RES_COMMENTS:

  Result comments

- PRJ_UID:

  Project unique ID

- MLOC_UID:

  Monitoring location unique ID
