# Example Designated Use (DU) Processed Table

Example dataset of processed designated use criteria for waterbodies.

## Usage

``` r
example_DU_processed
```

## Format

A data frame with 340 rows and 11 columns:

- WATER_ID:

  Assessment Unit identifier

- DU:

  Designated use category

- Waterbody:

  Waterbody type

- TN_SITE_CLASS:

  Total nitrogen site classification

- TP_SITE_CLASS:

  Total phosphorus site classification

- ECOREGION:

  Ecoregion classification

- ELEVATION:

  Elevation in feet

- Criteria_Name:

  Name of water quality criterion

- Criteria_Value:

  Numeric value of the criterion

- CHR_UID:

  Characteristic unique identifier

- CHR_UID_Unique:

  Unique identifier for characteristic and criterion combination

## Source

Generated by `Load_Example_Files()` in `data-raw`.
