# Example LANL DU Table

Example dataset containing designated use information for LANL sites.

## Usage

``` r
example_LANL_DU_table
```

## Format

A data frame with 15 rows and 25 columns:

- PRJ_NAME:

  Project name

- WATER_NAME:

  Waterbody name

- WATER_ID:

  Waterbody identifier

- STATION_NAME:

  Monitoring station name

- STATION:

  Monitoring station code

- DU:

  Designated use category

- WGROUP:

  Waterbody group

- WBODY:

  Waterbody type

- WQS_REFERENCE:

  Reference to water quality standards

- ECOLI_GEOMEAN:

  E. coli geometric mean value

- ECOLI_SINGLE:

  Single sample E. coli value

- TEMP_6T3:

  Temperature statistic (6T3)

- TEMP_4T3:

  Temperature statistic (4T3)

- TEMP_WQC:

  Temperature water quality criterion

- PH_LOW:

  Lower pH threshold

- PH_HIGH:

  Upper pH threshold

- DO_WQC:

  Dissolved oxygen water quality criterion

- SC_WQC:

  Specific conductance water quality criterion

- TP_WQC:

  Total phosphorus water quality criterion

- TDS_WQC:

  Total dissolved solids water quality criterion

- SO4_WQC:

  Sulfate water quality criterion

- CHL_WQC:

  Chlorophyll water quality criterion

- NMAC_NOTES:

  Notes from NMAC

- MLOC_UID:

  Monitoring location unique identifier

- PRJ_UID:

  Project unique identifier

## Source

Generated by `Load_Example_Files()` in `data-raw`.
