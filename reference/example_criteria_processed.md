# Example Criteria Processed Table

Example dataset of processed water quality criteria linked to designated
uses and characteristics.

## Usage

``` r
example_criteria_processed
```

## Format

A data frame with 1,321 rows and 12 columns:

- CHR_UID:

  Characteristic unique identifier

- CHR_NAME:

  Name of the characteristic (e.g., nitrate, temperature)

- FRACTION:

  Sample fraction (e.g., dissolved, total)

- DU:

  Designated use category

- Waterbody:

  Waterbody qualifier

- PERSISTENT:

  Indicator if characteristic is persistent (Y/N)

- HD_METAL:

  Indicator if characteristic is a heavy metal (Y/N)

- TOXIC:

  Indicator if characteristic is toxic (Y/N)

- TURBAP:

  Indicator for turbidity applicability (Y/N)

- UNITS:

  Units of measurement for the criterion

- Criteria_Type:

  Type of criterion (e.g., chronic, acute)

- Magnitude_Numeric:

  Numeric magnitude of the criterion
