# Example Criteria Table

Example dataset containing criteria for SQUID analysis.

## Usage

``` r
example_criteria_table
```

## Format

A data frame with 1042 rows and 28 columns:

- CHR_UID:

  Characteristic unique ID

- CHR_NAME:

  Characteristic name

- CAS:

  CAS number

- FRACTION:

  Fraction type

- DU:

  Designated use

- WBODY:

  Waterbody type

- WGROUP:

  Waterbody group

- ECOLI_SINGLE:

  Single sample E. coli criterion

- SC_WQC:

  Specific conductance criterion

- TEMP_WQC:

  Temperature criterion

- LT:

  Less than threshold indicator

- GT:

  Greater than threshold indicator

- HD_METAL:

  Hardness dependent metal indicator

- TOXIC:

  Toxicity indicator

- TURBAP:

  Turbidity applicability

- DWS:

  Drinking water supply indicator

- IRR:

  Irrigation indicator

- LW:

  Livestock watering indicator

- WH:

  Wildlife habitat indicator

- ACUTE:

  Acute toxicity criterion

- CHRONIC:

  Chronic toxicity criterion

- HH:

  Human health criterion

- PERSISTENT:

  Persistence indicator

- UNITS:

  Units of measurement

- ATTAINS CAUSE_GROUP_NAME:

  ATTAINS cause group name

- ATTAINS CAUSE_ID:

  ATTAINS cause identifier

- ADB_CAUSE_NO:

  ADB cause number

- Comments (SCB):

  Comments from SCB
